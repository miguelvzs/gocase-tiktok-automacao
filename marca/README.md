# Ativos de marca

`gocase.png` — logotipo exibido em fade nos últimos segundos do vídeo.

Requisitos: PNG **com transparência**, wordmark horizontal, pelo menos 600 px de
largura. O sistema redimensiona para cerca de 30% da largura do vídeo.

O arquivo não é versionado: é material de marca de terceiro, e o repositório é
público. Sem ele o vídeo é montado normalmente, sem assinatura, e o relatório
registra `logo: ausente`.
